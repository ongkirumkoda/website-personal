‚ö° LANGKAH CEPAT (5 MENIT)
=====================================

1. BACKUP FILE LAMA
   - Backup .htaccess
   - Backup .env
   - Backup index.php
   - Backup app/core/Router.php

2. COPY FILE BARU
   - siraga-fix/.htaccess ‚Üí siraga/.htaccess
   - siraga-fix/.env ‚Üí siraga/.env
   - siraga-fix/index.php ‚Üí siraga/index.php
   - siraga-fix/Router.php ‚Üí siraga/app/core/Router.php
   - siraga-fix/test.php ‚Üí siraga/test.php

3. EDIT KONFIGURASI
   
   A. Edit .htaccess, cari baris:
      RewriteBase /siraga/
      
      Sesuaikan dengan folder Anda:
      - Di http://localhost/siraga/ ‚Üí /siraga/
      - Di http://localhost/ ‚Üí /
   
   B. Edit .env, cari baris:
      APP_URL=http://localhost/siraga
      BASE_PATH=/siraga
      
      Sesuaikan dengan folder Anda:
      - Di http://localhost/siraga/ ‚Üí BASE_PATH=/siraga
      - Di http://localhost/ ‚Üí BASE_PATH=/
      
      Juga sesuaikan database:
      DB_NAME=siraga_db
      DB_USER=root
      DB_PASS=             ‚Üê Kosongkan jika XAMPP default

4. AKTIFKAN mod_rewrite
   
   A. Buka: C:\xampp\apache\conf\httpd.conf
   
   B. Cari dan HAPUS tanda # di depan:
      LoadModule rewrite_module modules/mod_rewrite.so
   
   C. Cari <Directory> dan ubah:
      AllowOverride None  ‚Üí  AllowOverride All
   
   D. Save dan RESTART Apache dari XAMPP

5. TEST SISTEM
   
   Buka browser:
   http://localhost/siraga/test.php
   
   Pastikan semua ‚úÖ hijau

6. AKSES WEBSITE
   
   http://localhost/siraga/
   
   Jika muncul homepage ‚Üí BERHASIL! üéâ
